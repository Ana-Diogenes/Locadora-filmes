# Locadora de filmes
Projeto final de Introdução a programação
***
Conteúdos vistos na disciplina aplicados no projeto:
* Manipulação de arquivos
* Uso de bibliotecas
* Estruturas de controle e repetição
* Organização do código
***
Armazenar dados temporariamente, permitindo:
* Inclusão
* Exclusão
* Consulta
* Alteração
* Gerenciar pelo menos 50 itens armazenados.
